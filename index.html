Simon Poole’s Online portfolio

University of Leeds GISc. Student

This is an online portfolio of work prepared for the MSc Geographical Information Science at Leeds University, Department of Geography.

Agent based model

This piece of work is an agent based model. Agents move around an environment (a 300x300 raster rather like
a digital elevation model) and interact with it by “eating” it as they move. It is a tournament and when they meet more
powerful agents they
will loose some of what they have eaten. At the end a “League Table” of agents is produced
showing the winning agent.
Its features include;

  • Agents named after English football teams
  • Agents enabled to “jump” away to flee more powerful agents
  • Animation
  • A simple GUI

“The main goal of the model was to develop python coding skills
from a basic level including key elements such as object
oriented programming (and re-use of code), webscraping and
in/out functionality”.

All the files can be found in the repo at https://github.com/simonhp66/simonhp66.github.io Please start with the ReadMe file which contains all instruction for using the model.
